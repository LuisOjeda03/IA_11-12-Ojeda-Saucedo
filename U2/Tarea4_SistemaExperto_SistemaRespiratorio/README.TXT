Sistema Experto - Sistema respiratorio
En este apartado se sube todo el proceso de desarrollo de un sistema experto capaz de ayudar con el diagnostico de diversas enfermedades respiratorias.

1. Primera parte: Se tiene un PDF con distinta información que se necesita sacada de distintas páginas con respaldo médico donde se recolectó información para su futura
utilización en la programación.

2. Segunda parte: Graficamos una Red de Inferencia para mostrar como como se organizan los síntomas con las distintas enfermedades con las operaciones logicas AND/OR y se elaboraron reglas escritas que representan lo anteriormente graficado y con una estructura que nos ayuda para manejar la logica proposicional.

3. Tercera parte: En este documento se utilizan las reglas creadas anteriormente y se crean con una lógica de predicados.

4. Cuarta parte: En esta carpeta se encuentra el código ejecutable del Sistema experto para el diagnostico de enfermedades respiratorias con su documentación.
