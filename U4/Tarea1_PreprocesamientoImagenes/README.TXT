Este proyecto consiste en cargar un dataset de imágenes clasificadas por emociones y aplicar técnicas de preprocesamiento usando Albumentations para mejorar la robustez del modelo ante variaciones en iluminación, rotación y escala.
Se utiliza FiftyOne para visualizar y explorar tanto el dataset original como el dataset aumentado.

# Nota: No realizamos un barrido de las imágenes ya que es una inmensa cantidad de ejemplos que se utilizan por cada emoción,
por lo tanto se prefirió omitirlo.

Página donde se obtuvo el dataset original:
https://www.kaggle.com/datasets/furcifer/fane-facial-expressions-and-emotion-dataset
